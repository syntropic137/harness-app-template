# Harness App Template

`harness-app-template` is a canonical starting point for projects that want the
environment around AI coding agents to be explicit, observable, and versioned.
The repository is the artifact: fork it or click **Use this template**, then run
the initialization recipe for your project name.

The template ships a polyglot workspace, a documented set of harness slots,
diff-scoped hooks, telemetry examples, and update tooling for pulling future
harness improvements without touching consumer-owned application code.
