# Code of Conduct

This project follows the [Contributor Covenant Code of Conduct, v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

By participating in this project — as a contributor, an issue reporter, or an AI coding agent operating on a contributor's behalf — you agree to abide by its terms.

## Reporting

If you witness or experience behavior that violates this code of conduct, please report it via the project's GitHub issue tracker (`https://github.com/syntropic137/harness-app-template/issues`) or by contacting a maintainer directly. Reports are treated with discretion.

## Scope

This code applies to all project spaces — issue trackers, pull requests, mailing lists, chat channels, and any space where individuals are acting on behalf of the project.
