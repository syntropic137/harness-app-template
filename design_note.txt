Design Note: create-harness-app npx wrapper

Objective:
Build create-harness-app as a standalone, npx-deployable Node.js CLI tool.

Core Flow:
1. Interactive Prompts: project name, dest dir, template variant (via 'prompts').
2. Fetch Template: Pull tarball directly from syntropic137/harness-app-template via GitHub API.
3. Variable Substitution: Regex replace {{PROJECT_NAME}} across downloaded files.
4. Initialization: Run git init.
5. Handoff: Print next steps (e.g., just bootstrap).

Tech Stack:
- Runtime: Node.js
- Interactivity: prompts
- Styling: kolorist
- File Ops: Native fetch + tar extraction, fs for variable subst.

Structure:
create-harness-app/
├── package.json (bin: ./bin/cli.js)
├── bin/cli.js
└── src/ (index.ts, prompts.ts, fetch.ts, transform.ts)
